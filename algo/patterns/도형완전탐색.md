> [!abstract] 핵심 요약
> 2D 배열 위의 고정 도형(트리오미노·테트로미노 등)을 모든 위치·방향에 대해 열거해 최적값을 구하는 패턴.
> 도형의 상대 좌표를 오프셋 배열로 정의하고, 경계 조건을 일괄 검사한다.

## 언제 쓰나

- 고정 크기 도형(L자, I자, T자 등)을 2D 배열에 올려 합/최솟값/최댓값 탐색
- 도형의 방향(회전·대칭)이 여러 개일 때 각각 열거
- 도형 종류가 적고 격자 크기가 작을 때 (완전탐색 허용)

## 핵심 아이디어

각 기준 셀 `(y, x)`를 순회하며, 도형의 나머지 셀들을 **상대 오프셋**으로 정의해 합산한다.
경계를 벗어나는 경우 삼항 연산자나 if-guard로 0 또는 무효값 처리.

## 복잡도

| 항목 | 복잡도 | 비고 |
|------|--------|------|
| 시간 | O(N·M·S) | S = 도형 방향 수 |
| 공간 | O(1) | 추가 배열 불필요 |

## Java 코드 템플릿 — 트로미노

```java
// L자형 트로미노 4방향 (기준 셀 포함 3칸)
for (int y = 0; y < N; y++) {
    for (int x = 0; x < M; x++) {
        // ┘ 오른쪽+위
        int s0 = (x+1 >= M || y-1 < 0) ? 0
                : board[y][x] + board[y][x+1] + board[y-1][x];
        // └ 왼쪽+위
        int s1 = (x-1 < 0  || y-1 < 0) ? 0
                : board[y][x] + board[y][x-1] + board[y-1][x];
        // ┐ 오른쪽+아래
        int s2 = (x+1 >= M || y+1 >= N) ? 0
                : board[y][x] + board[y][x+1] + board[y+1][x];
        // ┌ 왼쪽+아래
        int s3 = (x-1 < 0  || y+1 >= N) ? 0
                : board[y][x] + board[y][x-1] + board[y+1][x];
        answer = Math.max(answer, Math.max(s0, Math.max(s1, Math.max(s2, s3))));
    }
}

// 직선형 트로미노 2방향 (가로/세로 3칸)
for (int y = 0; y < N; y++) {
    for (int x = 0; x < M; x++) {
        int h = (x-1 < 0 || x+1 >= M) ? 0
                : board[y][x-1] + board[y][x] + board[y][x+1];
        int v = (y-1 < 0 || y+1 >= N) ? 0
                : board[y-1][x] + board[y][x] + board[y+1][x];
        answer = Math.max(answer, Math.max(h, v));
    }
}
```

## 경계 처리 패턴

```java
// 패턴 A: 삼항 연산자 (값이 0이어도 무방한 경우)
int sum = (out_of_bounds) ? 0 : board[y][x] + ...;

// 패턴 B: if-guard (continue/skip)
if (y-1 < 0 || x+1 >= M) continue;
int sum = board[y][x] + board[y][x+1] + board[y-1][x];
```

> [!note] board 크기 주의
> 문제의 행 크기와 열 크기가 다를 때(N≠M) `board = new int[N][M]`으로 선언한다.
> 경계 조건에서 행은 `< N`, 열은 `< M`을 각각 사용해야 한다.

## 주의사항 / 자주 하는 실수

> [!warning] 실수 포인트
> - 도형 방향을 일부 빠뜨림 — 종이에 도형을 그린 뒤 회전·대칭 모두 열거 확인
> - `board = new int[N][N]`으로 선언해 N≠M 케이스에서 런타임 에러
> - 경계 조건에서 행/열 한계값 혼용 (`>= N` 써야 할 곳에 `>= M`)
