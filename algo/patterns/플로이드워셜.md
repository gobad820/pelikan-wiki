> [!abstract] 핵심 요약
> 모든 정점 쌍 사이의 최단 거리를 O(N³)에 구하는 DP 기반 알고리즘.
> "경유 노드 k를 최외곽 루프"로 두는 것이 핵심.

## 언제 쓰나

- 모든 정점 쌍의 최단 거리가 필요할 때
- 정점 간 도달 가능 여부(연결성)를 한 번에 판별할 때
- N이 작을 때 (N ≤ 500 수준, O(N³) 허용 범위)
- 음수 가중치가 존재하지만 음수 사이클은 없을 때

## 핵심 아이디어

경유 노드 집합을 {1}, {1,2}, ..., {1,...,N}으로 점진적으로 확장한다.
k번 노드까지 경유 허용 시, i→j 최단거리는:

$$dist[i][j] = \min(dist[i][j],\ dist[i][k] + dist[k][j])$$

k가 최외곽 루프여야 하는 이유: `dist[i][k]`와 `dist[k][j]`는 이미 {1,...,k-1}만 경유한 최적값이어야 하므로, k 루프가 가장 먼저 확정되어야 한다.

## 복잡도

| 항목 | 복잡도 | 비고 |
|------|--------|------|
| 시간 | O(N³) | 3중 루프 |
| 공간 | O(N²) | 인접 행렬 |

## Java 코드 템플릿

```java
// 초기화
long[][] dist = new long[N + 1][N + 1];
final long INF = Long.MAX_VALUE / 2; // 오버플로우 방지
for (int i = 1; i <= N; i++) {
    Arrays.fill(dist[i], INF);
    dist[i][i] = 0;
}

// 간선 입력
dist[u][v] = weight; // 유향
dist[u][v] = dist[v][u] = weight; // 무향

// 플로이드 워셜 — k가 반드시 최외곽
for (int k = 1; k <= N; k++) {
    for (int i = 1; i <= N; i++) {
        if (dist[i][k] == INF) continue; // 가지치기
        for (int j = 1; j <= N; j++) {
            if (dist[k][j] == INF) continue;
            dist[i][j] = Math.min(dist[i][j], dist[i][k] + dist[k][j]);
        }
    }
}
```

## 주의사항 / 자주 하는 실수

> [!warning] 실수 포인트
> - **루프 순서 오류**: `i→j→k` 순서로 쓰면 틀린다. 반드시 `k→i→j`
> - **오버플로우**: `int` 사용 시 INF + INF가 음수가 됨 → `long` 또는 `INF/2` 사용
> - **answer 초기화 누락**: 다중 테스트케이스에서 `answer`를 매 TC마다 초기화해야 함
> - **자기 자신 거리**: `dist[i][i] = 0`으로 초기화 필수
