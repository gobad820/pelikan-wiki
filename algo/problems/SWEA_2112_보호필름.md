> [!info] 문제 정보
> - **플랫폼**: SWEA
> - **번호**: 2112
> - **유형**: 백트래킹, 완전탐색

## 문제 요약

D×W 크기의 필름이 있고 각 셀은 A(0) 또는 B(1) 성질을 가진다. 약품을 주입하면 한 행 전체가 같은 성질로 바뀐다. 내구성 검사를 통과하려면 각 열에 K개 이상 연속된 동일 성질이 있어야 한다. 최소 주입 횟수를 구한다.

## 접근법

> [!tip] 핵심 접근
> 각 행에 대해 스킵/약물A/약물B 3가지를 재귀 탐색. cnt >= answer이면 즉시 가지치기.

### 사고 흐름

1. 행 단위 3분기 백트래킹: 스킵(cnt 유지) → 약물A → 약물B
2. 스킵을 먼저 탐색 → answer를 빠르게 갱신 → 이후 가지 많이 제거
3. 내구성 검사: 슬라이딩 윈도우로 K개 연속 동일값 여부 확인
4. 주입 전 이미 통과 시 answer = 0으로 바로 반환

### 핵심 코드

```java
void injectChemicals(int idx, int cnt) {
    if (answer <= cnt) return;                    // 가지치기
    if (checkFragility(board)) { answer = cnt; return; }
    if (idx == D) return;

    injectChemicals(idx + 1, cnt);                // 스킵 (먼저!)

    int[] backup = board[idx].clone();            // 값 복사 필수
    Arrays.fill(board[idx], 0);                  // 약물A
    injectChemicals(idx + 1, cnt + 1);
    Arrays.fill(board[idx], 1);                  // 약물B
    injectChemicals(idx + 1, cnt + 1);
    board[idx] = backup;                          // 복원
}

boolean checkFragility(int[][] board) {
    for (int col = 0; col < W; col++) {
        boolean colOk = false;
        for (int row = 0; row <= D - K; row++) {
            int sum = 0;
            for (int l = 0; l < K; l++) sum += board[row + l][col];
            if (sum == 0 || sum == K) { colOk = true; break; }
        }
        if (!colOk) return false;
    }
    return true;
}
```

## 복잡도

| 항목 | 복잡도 | 근거 |
|------|--------|------|
| 시간 | O(3^D × D×W×K) 추정 | D≤13, 가지치기로 실제 탐색 공간은 훨씬 작음 |
| 공간 | O(D×W) | board 배열 + 재귀 스택 |

## 배운 점 / 인사이트

> [!note] 기억할 것
> - 스킵을 먼저 탐색하는 순서가 가지치기 효율을 높임
> - `board[idx].clone()` 으로 값 복사 후 `board[idx] = backup` 으로 복원
> - `checkFragility` 내 열 통과 즉시 `break` 추가로 불필요한 반복 제거
