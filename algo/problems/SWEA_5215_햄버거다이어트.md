# SWEA 5215 - 햄버거 다이어트

## 문제 요약
N개 재료, 각각 맛(S)과 칼로리(C). 칼로리 상한 L 이하로 재료를 골라 맛의 합 최대화.
각 재료는 최대 1번 사용.

## 변수 매핑 (핵심)
| 0/1 Knapsack 용어 | 이 문제 |
|-------------------|---------|
| 무게(Weight) | 칼로리 C |
| 가치(Value) | 맛 S |
| 배낭 용량 K | 칼로리 상한 L |

→ 변수만 치환하면 **그대로 0/1 Knapsack** 패턴 적용 가능.

## 풀이 코드
```java
dp = new int[L + 1];
for (int i = 1; i <= N; i++) {
    // S = 맛(가치), C = 칼로리(무게)
    for (int j = L; j >= C; j--) {
        dp[j] = Math.max(dp[j], dp[j - C] + S);
    }
}
answer = dp[L];
```

## 핵심 포인트
- 문제 표현이 달라도 **변수 매핑**만 올바르면 Knapsack 틀 그대로 사용
- 역순 순회로 중복 선택 방지

## 복잡도
- 시간: O(N × L)
- 공간: O(L)
