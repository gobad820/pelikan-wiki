# 백준 11726 - 2×n 타일링

> 2×n 직사각형을 2×1 타일로 채우는 경우의 수 (mod 10007)

## 핵심 아이디어

n번째 열을 채우는 방법은 두 가지뿐:
1. **세로 타일 1개** → 나머지 2×(n-1)을 채우는 경우 = `f(n-1)`
2. **가로 타일 2개** → 나머지 2×(n-2)를 채우는 경우 = `f(n-2)`

→ 피보나치 수열과 동일한 점화식

## 점화식

```
dp[n] = (dp[n-1] + dp[n-2]) % 10007
```

## 기저값

```
dp[1] = 1   // 세로 타일 1개만 가능
dp[2] = 2   // 세로 2개 | 가로 2개
```

## Java 코드

```java
static int[] dp = new int[1010];

static {
    dp[1] = 1;
    dp[2] = 2;
    for (int i = 3; i < 1010; i++) {
        dp[i] = (dp[i - 1] + dp[i - 2]) % 10007;
    }
}
// dp[N]이 정답
```

> `static {}` 블록 전처리 → 쿼리 O(1)

## 시간복잡도

- 전처리: O(N)
- 쿼리: O(1)

## 주의사항

- **mod 10007** 누락 주의: 매 단계 `% 10007` 적용
- `dp[0]`은 사용하지 않음 (1-indexed)

## 피보나치와의 관계

| | 피보나치 | 2×n 타일링 |
|--|--|--|
| base | dp[1]=1, dp[2]=1 | dp[1]=1, dp[2]=2 |
| 점화식 | dp[i] = dp[i-1] + dp[i-2] | dp[i] = dp[i-1] + dp[i-2] |
| 차이 | dp[2]=1 | dp[2]=2 |

초기값만 다르고 점화식은 동일.
