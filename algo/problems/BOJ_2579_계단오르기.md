# 백준 2579 - 계단 오르기

> N개의 계단을 오를 때 최대 점수 수집
> - 연속으로 3개 계단을 밟으면 안 됨
> - 마지막 계단은 반드시 밟아야 함
> - 한 번에 1칸 또는 2칸 이동 가능

## 핵심 아이디어

**"연속 몇 번째 계단인가"를 상태로** 표현.

- `dp[i][0]` = i번째 계단이 연속으로 **1번째** 밟히는 경우의 최대 점수
- `dp[i][1]` = i번째 계단이 연속으로 **2번째** 밟히는 경우의 최대 점수

3번째 연속은 불가하므로 상태가 2개면 충분.

## 점화식

```
dp[i][0] = max(dp[i-2][0], dp[i-2][1]) + stair[i]
           // i를 밟을 때 i-1을 건너뜀 → i는 연속 1번째

dp[i][1] = dp[i-1][0] + stair[i]
           // i-1이 연속 1번째여야 i가 연속 2번째 가능
           // (i-1이 연속 2번째면 → i는 연속 3번째가 되어 불가)
```

## 기저값

```
dp[1][0] = stair[1]   // 1번 계단만 밟음 (연속 1번째)
dp[1][1] = 0          // 불가 (1번 계단이 연속 2번째가 되려면 0번 계단이 존재해야 함)
dp[2][0] = stair[2]   // 1번 건너뛰고 2번 밟음 (연속 1번째)
dp[2][1] = stair[1] + stair[2]  // 1번, 2번 연속 밟음
```

## 정답

```
max(dp[N][0], dp[N][1])
```

## Java 코드

```java
int N = Integer.parseInt(br.readLine());
int[] stair = new int[N + 1];
for (int i = 1; i <= N; i++) stair[i] = Integer.parseInt(br.readLine());

int[][] dp = new int[N + 1][2];
dp[1][0] = stair[1];
dp[1][1] = 0;
if (N >= 2) {
    dp[2][0] = stair[2];
    dp[2][1] = stair[1] + stair[2];
}

for (int i = 3; i <= N; i++) {
    dp[i][0] = Math.max(dp[i-2][0], dp[i-2][1]) + stair[i];
    dp[i][1] = dp[i-1][0] + stair[i];
}

System.out.println(Math.max(dp[N][0], dp[N][1]));
```

## 시간복잡도

- O(N)

## 주의사항

- N=1일 때 `dp[2]` 접근 → 범위 체크 필요
- `dp[i][1] = dp[i-1][1] + stair[i]` 로 쓰면 틀림 (연속 3번째 허용됨)
- 마지막 계단 반드시 밟아야 → 정답이 `max(dp[N][0], dp[N][1])`이지, 전체 max가 아님
