## 오늘 배운 패턴/개념
- [[Backtracking]] - 순열 4종 세트: 중복순열, 순열, 중복조합, 조합

## 핵심 코드
```java
// 순열 (isSelected[] 방식)
void dice2(int depth) {
    if (depth == n) { ... return; }
    for (int i = 1; i <= 6; i++) {
        if (isSelected[i]) continue;
        isSelected[i] = true;
        numbers[depth] = i;
        dice2(depth + 1);
        isSelected[i] = false; // 백트래킹!
    }
}

// 조합 (start 포인터 방식)
void dice4(int depth, int start) {
    if (depth == n) { ... return; }
    for (int i = start; i <= 6; i++) {
        numbers[depth] = i;
        dice4(depth + 1, i + 1); // 다음은 i+1부터
    }
}
```

## 문제 풀이
- DiceTest: 주사위(1~6) N번 던지기 - 4가지 모드 구현
- PermuatationInputTest: 입력 배열에서 nPr 순열

## 오늘의 인사이트
- 4종 차이: 중복 허용 여부 × 순서 고려 여부
  - 중복순열: 매번 1~N 전체 선택 가능
  - 순열: `isSelected[]`로 중복 방지
  - 중복조합: `start` 유지 (현재 원소 재선택 가능)
  - 조합: `start = i+1` (현재 이후만)
- 백트래킹의 핵심: **재귀 후 상태 복원** (`isSelected[i] = false`)
