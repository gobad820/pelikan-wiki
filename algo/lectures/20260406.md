# 강의 노트 - 2026-04-06

## 오늘 배운 패턴/개념
- [[DP_Memoization]] — 재귀 + 캐시, Top-Down DP
- [[DP_Tabulation]] — 반복문으로 Bottom-Up DP

## 핵심 코드

```java
// Top-Down: 재귀 + 캐시
static long fibo(int n) {
    if (n < 2) return n;
    if (dp[n] != -1) return dp[n];
    return dp[n] = fibo(n-1) + fibo(n-2);
}

// Bottom-Up: static 블록으로 전처리
static {
    dp[1] = 1; dp[2] = 1;
    for (int i = 3; i <= 100; i++) dp[i] = dp[i-1] + dp[i-2];
}
```

## 문제 풀이
- **FibonacciTest**: Top-Down + Bottom-Up 비교 실습
  - `call[]` 배열로 각 n의 호출 횟수 추적 → 메모이제이션 효과 확인
  - `totalCount`로 전체 함수 호출 수 측정

## 오늘의 인사이트
- `dp` 배열 초기화는 `-1` (0이 유효한 값일 수 있음)
- Bottom-Up은 `static {}` 블록 활용 → 클래스 로딩 시 전처리, 이후 O(1) 쿼리
- `call[n]`을 출력해보면 메모이제이션 효과를 시각적으로 확인 가능
