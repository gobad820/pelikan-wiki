## 오늘 배운 패턴/개념
- [[Backtracking]] - 비트마스크 순열 + 다음 순열(Next Permutation)

## 핵심 코드
```java
// 비트마스크 순열
void permutation(int depth, int flag) {
    if (depth == r) { ... return; }
    for (int i = 0; i < n; i++) {
        if ((flag & (1 << i)) != 0) continue; // i번째 이미 선택됨
        numbers[depth] = input[i];
        permutation(depth + 1, flag | (1 << i)); // i번째 비트 켜기
    }
}

// 다음 순열
boolean nextPermutation() {
    int i = arr.length - 1;
    while (i > 0 && arr[i-1] >= arr[i]) i--; // 뒤에서 감소 구간 찾기
    if (i == 0) return false;
    int j = arr.length - 1;
    while (arr[i-1] >= arr[j]) j--;           // i-1 자리에 올 값 찾기
    swap(i-1, j);
    // i 이후 역순 정렬 (내림차순 → 오름차순)
    for (int k = arr.length - 1; i < k; i++, k--) swap(i, k);
    return true;
}
```

## 오늘의 인사이트
- 비트마스크 순열: `boolean[]` 대신 int 1개로 선택 상태 관리 → 재귀 파라미터로 넘김
- 다음 순열 3단계: ① 감소 구간 찾기 → ② swap → ③ i 이후 역전
- `Arrays.sort()` 후 `do { } while(nextPermutation())` 패턴으로 전체 순열 순회
