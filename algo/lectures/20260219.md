## 오늘 배운 패턴/개념
- [[Graph_AdjList]] + [[BFS_DFS]] - ArrayList 인접 리스트 + BFS 구현

## 핵심 코드
```java
List<Integer>[] adjList = new List[V + 1]; // 1-indexed
for (int i = 0; i <= V; i++) adjList[i] = new ArrayList<>();

adjList[from].add(to);
adjList[to].add(from);

// BFS
void bfs(int start) {
    Deque<Integer> dq = new ArrayDeque<>();
    boolean[] vis = new boolean[V];
    vis[start] = true;
    dq.offerLast(start);
    while (!dq.isEmpty()) {
        int cur = dq.pollFirst();
        for (int next : adjList[cur]) {
            if (vis[next]) continue;
            dq.offerLast(next);
            vis[next] = true; // 큐에 넣을 때 방문 표시!
        }
    }
}
```

## 오늘의 인사이트
- `new List[V+1]`: Java 제네릭 배열 경고 발생하지만 관용적으로 사용
- `vis[next] = true`는 **offerLast 직전**에 — poll 시점에 하면 중복 삽입됨
- day0213(연결 리스트)보다 ArrayList가 코드가 간결하고 실수가 적음
