## 오늘 배운 패턴/개념
- [[TwoPointer]] - 투 포인터: 양끝 포인터와 슬라이딩 윈도우 두 패턴

## 핵심 코드
```java
// 패턴1: 두 수의 합 (양끝 포인터)
Arrays.sort(arr);
int s = 0, e = n - 1;
while (s < e) {
    int sum = arr[s] + arr[e];
    if (sum == target) { ++ans; ++s; --e; }
    else if (sum < target) ++s;
    else --e;
}

// 패턴2: 구간 합 = K
int start = 0, end = 0, sum = 0, cnt = 0;
while (end < n) {
    if (sum >= k) { if (sum == k) ++cnt; sum -= arr[start++]; }
    else sum += arr[end++];
}
```

## 문제 풀이
- TwoPointerTest1: 정렬된 배열에서 두 수 합 = target인 쌍 개수
- TwoPointerTest2: 양수 배열에서 구간 합 = K인 구간 개수

## 오늘의 인사이트
- 투 포인터는 O(N²) 완전탐색을 O(N)으로 줄이는 핵심 기법
- `sum >= k`일 때 먼저 처리하고 start 이동: 순서가 틀리면 카운팅 누락
