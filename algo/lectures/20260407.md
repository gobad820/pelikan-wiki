## 오늘 배운 패턴/개념
- DP_Tabulation - 파스칼 삼각형: 2D 테이블 DP 설계 연습
- DP 설계 7단계 프레임워크 학습

## DP 설계 7단계 (핵심 메모)
```
1. 문제 이해 - 최적해? 경우의 수?
2. 부분 문제 식별
3. 상태 표현 - D[i] 또는 D[i][j] 설계
4. 점화식 도출 - D[i]와 D[i-1]의 관계 파악 + base case 찾기
5. 테이블 생성 + base case 초기화 → 점화식으로 채우기
6. 해 도출 - 마지막 값이 무조건 답이 아님, 문제가 원하는 상태 값 찾기
7. (선택) 공간 복잡도 최적화
```

## 핵심 코드
```java
// 파스칼 삼각형 - 2D Bottom-Up DP
// 점화식: dp[i][j] = dp[i-1][j-1] + dp[i-1][j]
// base case: dp[i][0] = 1, dp[i][i] = 1
int[][] dp = new int[N + 1][N + 1];
for (int i = 0; i <= N; i++) {
    dp[i][0] = 1;
    dp[i][i] = 1;
    for (int j = 1; j < i; j++) {
        dp[i][j] = dp[i-1][j-1] + dp[i-1][j];
    }
}
// dp[n][k] = nCk (이항계수)
```

## 오늘의 인사이트
- DP 설계에서 "마지막에 채워진 값이 무조건 해가 아님" — 문제가 원하는 상태 확인 필수
- 파스칼 삼각형 → 이항계수 테이블 → 순열/조합 빠른 계산 가능
- Bottom-Up은 하향식(재귀) 뒤집어서 생각하는 것에서 출발
