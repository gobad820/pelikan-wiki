## 오늘 배운 패턴/개념
- [[MST_Kruskal]] - 간선 정렬 + Union-Find로 최소 신장 트리 구성
- [[UnionFind]] - Kruskal의 사이클 감지 기반

## 핵심 코드
```java
static class Edge implements Comparable<Edge> {
    int start, end, weight;
    public int compareTo(Edge o) {
        return Integer.compare(this.weight, o.weight);
    }
}

// Kruskal 메인 로직
Arrays.sort(edgeList); // 가중치 오름차순 정렬
makeSets();

int cnt = 0, result = 0;
for (Edge edge : edgeList) {
    if (union(edge.start, edge.end)) { // 사이클 없으면 MST에 추가
        result += edge.weight;
        if (++cnt == V - 1) break;  // V-1개 간선 → 완성
    }
}
```

## 문제 풀이
- **최소 스패닝 트리 (Solution_3124)**: Kruskal 직접 적용
- **하나로 (Solution_1251)**: 가중치 계산이 특수한 MST (두 정점 좌표 거리 기반)
- **미생물 격리 (Solution_2382)**: 시뮬레이션 문제 (MST와 별개)

## 오늘의 인사이트
- Kruskal은 **Greedy** — 가장 가중치 작은 간선부터 탐욕적으로 선택
- `union()` 반환 `false` = 사이클 → 해당 간선 스킵
- `V-1`개 간선 선택 시 즉시 break → 불필요한 탐색 제거
- `Comparable` 구현으로 `Arrays.sort()` 직접 활용
