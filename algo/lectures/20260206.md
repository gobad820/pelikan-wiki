## 오늘 배운 패턴/개념
- [[Backtracking]] - 부분집합: 합이 targetSum인 부분집합 개수

## 핵심 코드
```java
// 부분집합 - 포함/미포함 두 분기
void subset(int cnt) {
    if (cnt == n) {
        // isSelected[] 상태 처리
        return;
    }
    isSelected[cnt] = false; subset(cnt + 1); // 미포함
    isSelected[cnt] = true;  subset(cnt + 1); // 포함
}

// 부분집합 합 계산 (파라미터로 합 누적)
void makeSubsetSum(int cnt, int pickCnt, int sum) {
    if (cnt == n) {
        if (pickCnt > 0 && sum == targetSum) totalCnt++;
        return;
    }
    makeSubsetSum(cnt + 1, pickCnt + 1, sum + input[cnt]); // 포함
    makeSubsetSum(cnt + 1, pickCnt, sum);                  // 미포함
}
```

## 문제 풀이
- SubsetTest: 합이 targetSum인 부분집합 개수 세기

## 오늘의 인사이트
- 부분집합은 각 원소에 대해 **포함/미포함 2가지 분기** → 2^N 가지
- 합 조건은 파라미터로 누적해서 내려가면 배열 순회 없이 O(1) 체크 가능
- `pickCnt > 0` 조건: 공집합 제외할 때 필요
