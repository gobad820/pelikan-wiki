## 오늘 배운 패턴/개념
- [[DivideAndConquer]] - 쿼드트리: 2D 배열 4분할 재귀

## 핵심 코드
```java
void cut(int r, int c, int size) {
    int sum = 0;
    for (int i = r; i < r + size; i++)
        for (int j = c; j < c + size; j++)
            sum += spaces[i][j];

    if (sum == size * size) { ++green; return; } // 전부 1
    if (sum == 0)           { ++white; return; } // 전부 0

    int half = size / 2;
    cut(r, c, half);           cut(r, c + half, half);
    cut(r + half, c, half);    cut(r + half, c + half, half);
}
```

## 문제 풀이
- MakeSpaceTest: N×N 격자를 쿼드트리로 분할, 흰색/초록색 구역 수 출력

## 오늘의 인사이트
- `sum == size * size` → 전부 1 체크 (합 = 넓이)
- 2D 범위: `(r, c)` ~ `(r+size-1, c+size-1)` → 반복문 `< r+size`
- 4분할: (r,c), (r, c+half), (r+half, c), (r+half, c+half)
