## 오늘 배운 패턴/개념
- Dijkstra - 단순 배열 + visited 스캔 방식 다익스트라 (O(V²))

## 핵심 코드
```java
Arrays.fill(minDist, Integer.MAX_VALUE);
minDist[start] = 0;

for (int i = 0; i < V; i++) {
    // Step1: 미방문 정점 중 minDist 최솟값 정점 선택
    int cur = -1, min = Integer.MAX_VALUE;
    for (int j = 0; j < V; j++) {
        if (!visited[j] && min > minDist[j]) { cur = j; min = minDist[j]; }
    }
    if (cur == -1) break;
    visited[cur] = true;

    // Step2: cur 경유 경로로 minDist 갱신
    for (Node temp = adjList[cur]; temp != null; temp = temp.next) {
        if (!visited[temp.to] && minDist[temp.to] > min + temp.weight)
            minDist[temp.to] = min + temp.weight;
    }
}

// 결과
int result = minDist[end] != Integer.MAX_VALUE ? minDist[end] : -1;
```

## 문제 풀이
- **Dijkstra_AdjList**: start~end 최단 경로 + 전체 minDist 배열 출력
- **Main_B_1753_최단경로**: 백준 1753 — 단일 출발 최단 경로

## 오늘의 인사이트
- 단순 배열 스캔 방식: O(V²) — 정점 수가 적을 때 PQ보다 구현이 단순
- `minDist` 배열 초기화: `Integer.MAX_VALUE` 사용 시 더하기에서 오버플로우 주의
- 도달 불가: `minDist[end] == Integer.MAX_VALUE` → `-1` 출력
