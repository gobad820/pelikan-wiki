## 오늘 배운 패턴/개념
- [[BFS_DFS]] - 완전 이진 트리 BFS: 배열 인덱스 규칙 + 레벨 구분 패턴

## 핵심 코드
```java
// 완전 이진 트리 레벨 BFS
int breadth = 0;
Deque<Integer> q = new ArrayDeque<>();
q.offerLast(root);
while (!q.isEmpty()) {
    int size = q.size(); // 현재 레벨 노드 수
    while (--size >= 0) {
        int cur = q.pollFirst();
        // 자식: cur*2, cur*2+1 (1-indexed)
        if (cur * 2 <= last) q.offerLast(cur * 2);
        if (cur * 2 + 1 <= last) q.offerLast(cur * 2 + 1);
    }
    ++breadth;
}
```

## 문제 풀이
- CompleteBinaryTree: 배열로 표현된 완전 이진 트리를 레벨 순서로 BFS

## 오늘의 인사이트
- 완전 이진 트리 1-indexed: 자식 `i*2`, `i*2+1`, 부모 `i/2`
- 레벨 구분: `int size = q.size()` 스냅샷 후 그 수만큼 처리
- `vis[next] = true`는 큐에 넣을 때 (poll 시점 X) — 중복 삽입 방지
