## 오늘 배운 패턴/개념
- [[Graph_AdjList]] - 연결 리스트로 구현하는 인접 리스트

## 핵심 코드
```java
class Node {
    int vertex;
    Node next;
    Node(int vertex, Node next) { this.vertex = vertex; this.next = next; }
}

Node[] adjList = new Node[V]; // null 초기화

// 간선 추가 (헤드 삽입 - O(1))
adjList[from] = new Node(to,   adjList[from]);
adjList[to]   = new Node(from, adjList[to]);

// 순회
for (Node cur = adjList[v]; cur != null; cur = cur.next) { ... }
```

## 오늘의 인사이트
- 헤드 삽입 방식: 새 노드가 기존 헤드를 next로 가짐 → O(1) 삽입
- 결과적으로 입력 역순으로 저장됨 (순서 중요한 문제 주의)
- `Node[V]` 초기화: Java 배열은 참조형이므로 자동으로 null
